Features

Focused on the tools that are currently available in the application.

📦 App Installer

Install popular Windows applications from a curated winget-powered catalog.

  • Search and select apps
  • Install selected apps in bulk
  • Check installed and upgradable packages
  • List and grid views
  • Export and import selected app lists

🔧 System Maintenance

Run common cleanup, repair, and diagnostics actions from one place.

  • Clean temporary files and cache
  • Empty Recycle Bin and cleanup Windows Update cache
  • Run SFC and DISM repair tools
  • Flush DNS, renew IP, and reset network stack
  • Launch Patch My PC for app updates

🧹 Debloat

Download and launch Windows debloat and optimization tooling when you need deeper cleanup.

  • Sparkle debloat integration
  • Windows bloatware cleanup workflow
  • Performance and privacy optimization focus
  • Administrator prompts only when required

♫ Spicetify

Customize Spotify with Spicetify tools directly from the app.

  • Install Spicetify
  • Uninstall Spicetify
  • Full Spotify and Spicetify uninstall option
  • Live progress output for long-running tasks

⚙ Windows Utilities

Access advanced Windows helper workflows without digging through menus.

  • Chris Titus Tech WinUtil launcher
  • Restart directly into BIOS/UEFI
  • Windows activation helper
  • Auto-login helper

🔄 Updates and UI

Modern Electron interface with automatic update support.

  • Automatic update checks
  • Installer, portable, and zip builds
  • Dark and light theme support
  • English and Greek interface
  • Optional settings sync when sign-in is configured

How to Use

Download, open the app, and choose the tool you need from the sidebar.

1

Download

Get the installer or portable version from the latest release.

2

Choose a Tool

Open Install Apps, System Maintenance, Debloat, Spicetify, BIOS, or Activation.

3

Run Actions

Select the action you want and follow any Windows permission prompts.

4

Stay Updated

Use the built-in updater to keep the app current.

Some tools make system-level changes and may require administrator rights. Review what each action does before running it.

Privacy

Make Your Life Easier does not collect personal data for tracking, advertising, telemetry, or analytics.

🔒 No Personal Data Collection

The app does not collect personal data, does not run ads, and does not include analytics tracking.

💻 Local-First Tools

Installer, maintenance, debloat, BIOS, activation, and Spicetify actions run on your own Windows device.

☁ Optional Sign-In

If cloud sync is enabled, sign-in is optional and used only for account session and settings sync. The app works without signing in.
